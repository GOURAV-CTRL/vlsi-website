Frequency Division by Even Numbers 🔁

All About VLSI · Lecture Notes — Part 17: Behavioural Modelling, Counter Method
The Idea

To divide a clock frequency by an even number N, use a free-running counter that counts input clock edges and toggles the output every N/2 edges. Because the output toggles twice per full count of N, the output period becomes exactly N times the input period.

This works cleanly for even N (2, 4, 6, ...) because N/2 is a whole number — no half-cycle correction is needed, unlike the odd-divide case.

Behavioural Verilog — Counter Method
module freq_div_even #( parameter N = 6 // Division factor: even number (2, 4, 6, ...) )( input wire clk_in, input wire rst_n, output reg clk_out ); integer count; always @(posedge clk_in or negedge rst_n) begin if (!rst_n) begin count <= 0; clk_out <= 0; end else begin if (count == (N/2 - 1)) begin clk_out <= ~clk_out; count <= 0; end else begin count <= count + 1; end end end endmodule

Non-blocking (<=) is used throughout since this is a clocked, sequential always block — matching the golden rule from Part 16.

Why the Counter Threshold is N/2 − 1
  • count increments once per input clock edge, starting from 0.
  • When count reaches N/2 − 1, that's the N/2-th edge counted (0, 1, 2, ... N/2−1) — so the output toggles.
  • Counter resets to 0 and the process repeats, giving a toggle exactly every N/2 input cycles.

Testbench Snippet
module tb_freq_div_even; reg clk_in, rst_n; wire clk_out; freq_div_even #(.N(4)) dut ( // change N to test 2, 4, 6 .clk_in(clk_in), .rst_n(rst_n), .clk_out(clk_out) ); always #5 clk_in = ~clk_in; // 100 MHz input clock initial begin clk_in = 0; rst_n = 0; #12 rst_n = 1; #200 $finish; end initial $monitor("T=%0t clk_in=%b clk_out=%b", $time, clk_in, clk_out); endmodule
Important Note
⚠️ This Method Only Works Directly for Even N

For odd division (e.g., divide-by-3, divide-by-5), a single posedge-triggered toggle counter can't produce a 50% duty cycle output — that needs a combined posedge + negedge approach, covered separately.
