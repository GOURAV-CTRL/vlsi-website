Divide-by-3 with True 50% Duty Cycle ⚖️

All About VLSI · Lecture Notes — Part 19: Posedge + Negedge Combination Trick
The Trick: Combine Posedge and Negedge Logic

To get an exact 50% duty cycle for odd division, the classic technique creates two divided clocks — one from a posedge-triggered counter, one from a negedge-triggered counter — then XOR's them together. Each half-clock counter individually gives a skewed duty cycle, but combining a posedge-based and negedge-based version cancels the skew.

Behavioural Verilog — Divide-by-3, 50% Duty Cycle
module freq_div_by3_50 ( input wire clk_in, input wire rst_n, output wire clk_out ); reg [1:0] cnt_pos, cnt_neg; reg clk_pos, clk_neg; // Posedge-triggered counter/toggle always @(posedge clk_in or negedge rst_n) begin if (!rst_n) begin cnt_pos <= 0; clk_pos <= 0; end else begin if (cnt_pos == 2) begin cnt_pos <= 0; clk_pos <= ~clk_pos; end else cnt_pos <= cnt_pos + 1; end end // Negedge-triggered counter/toggle always @(negedge clk_in or negedge rst_n) begin if (!rst_n) begin cnt_neg <= 0; clk_neg <= 0; end else begin if (cnt_neg == 2) begin cnt_neg <= 0; clk_neg <= ~clk_neg; end else cnt_neg <= cnt_neg + 1; end end // Combine both halves for exact 50% duty cycle assign clk_out = clk_pos ^ clk_neg; endmodule

The XOR combination is the standard divide-by-3 50%-duty trick: each internal signal (clk_pos, clk_neg) toggles every 3 edges of its own reference edge, and XOR-ing the two staggered waveforms produces a perfectly even high/low split at the output.

Why This Works — Intuition
  • clk_pos toggles every 3 rising edges of clk_in.
  • clk_neg toggles every 3 falling edges of clk_in — offset by half a clk_in period from clk_pos.
  • XOR-ing two signals of the same divided frequency but offset in phase produces a waveform whose high/low durations average out to exactly 50%.
⚠️ This Generalizes Beyond N = 3

The same posedge/negedge + XOR structure extends to any odd N by changing the toggle threshold (N-1 instead of 2) in both counters — the dual-edge cancellation principle stays the same.
