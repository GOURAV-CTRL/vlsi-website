Logic Gates & Universal Gates

Logic gates are the physical building blocks that implement Boolean operations in silicon. Understanding their truth tables — and why NAND and NOR are called universal gates — is essential before moving into combinational circuit design.

2Why NAND and NOR are "Universal"

A gate is called universal if it can be used, all by itself, to construct every other basic gate — AND, OR, and NOT included. NAND and NOR are the only two gates with this property.

NOT using NAND
Tie both inputs of a NAND gate together: (A · A)' = A'
AND using NAND
NAND followed by a NOT (itself built from NAND): (A·B)' then invert again → A·B
OR using NAND
Invert both inputs first, then NAND them: A' NAND B' = (A'·B')' = A + B  (De Morgan's)

The same three constructions apply symmetrically with NOR gates — invert-and-combine patterns built entirely from NOR give NOT, OR, and AND.

Why it matters in real chips — Fabricating a chip using only one type of gate simplifies the manufacturing library and standard-cell design. This is exactly why NAND/NOR-based logic is so common in real ASIC standard cell libraries.

3XOR & XNOR

XOR behaves like a "difference detector" — it's 1 only when its inputs disagree — making it the core building block of adders and parity checkers. XNOR is simply its complement, acting as an equality checker.

Expressions
A ⊕ B = A·B' + A'·B
(A ⊕ B)' = A·B + A'·B'
Key takeaway — NAND and NOR are the workhorses of real digital hardware because any circuit — no matter how complex — can be built entirely out of just one of these gates, which is a huge advantage for fabrication and standard-cell libraries.
